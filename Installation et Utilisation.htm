Installation et Utilisation de l'outil Duevera Formulaire

****************************************************************

****************************************************************

****************************************************************

Installation

L’installation de Duevera formulaire est simple : il n’y a pas de fichier à modifier " à la main " avec des variables ésotériques.
1. Décompactez sur votre ordinateur personnel le dossier " duevera_formulaire " contenant l’ensemble des fichiers de l’outil Duevera Formulaire.
2. Installez le contenu de ce dossier sur votre site (par FTP, comme vous le faites habituellement pour installer vos pages sur votre site).
3. Connectez vous avec votre navigateur sur votre site, dans un fichier intitulé " db.php ". Cliquez sur le bouton " Modifier les paramètres " et définissez les variables de connexion.

 

Configuration nécessaire

Vous devez disposer d’un hébergement Web avec :

  1. Un accès FTP pour l’installation des fichiers

  2. Le support de PHP3

  3. Un accès à une base de données MySQL.

Avant l’installation, vous devez avoir une base mySQL disponible. Sur de très nombreux hébergements, il faut soit demander l’activation d’une base mySQL à l’administrateur, soit suivre une procédure automatique en ligne (dans tous les cas, l’activation de la base mySQL n’a rien à voir avec Duevera Formulaire ; si vous avez des difficultés, seul votre hébergeur peut vous fournir les mots de passe nécessaires et vous expliquer comment activer votre compte mySQL).

Vous devez connaître les données de votre connexion MySQL (fournies par l’hébergeur) :

  • L’adresse de la base MySQL : par exemple sql.free.fr, ou localhost, ou vide

  • Votre login MySQL : souvent le même login que votre compte Web

  • Votre password MySQL : souvent le même que le compte Web



Modifier la configuration

Vous pouvez modifier la configuration de base de PHP dans le fichier php.ini.

Attention, dans ce fichier, les lignes qui commencent par un point virgule sont de commentaires.

Il faut modifier les lignes qui ne sont pas des commentaires, donc plus bas dans le fichier...

Vous pourrez modifier les lignes suivantes pour obtenir :

register_globals = On

magic_quotes_gpc = On

error_reporting = E_ALL & ~E_NOTICE

display_errors = On


Utilisation

L’utilisation est particulièrement simple si vous avez des notions de base de données. Cet outil vous permettra de gérer les tables (recherches paramétrés, insertion et suppression d’enregistrement… Sans faire aucune requête SQL).
